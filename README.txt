This model demonstrates Hybrid Robust Sliding Mode and PI Controller for Stand-alone Object's Power Supply.

To start the model:
1. Open file parameters.m
2. Set the values or use their defaults: K=100; Ts=1e-6; Wsm=1; Wpi=1;
3. Run file parameters.m
4. Observe the voltage and current waveforms in model file power_supply.slx
